xCtl -- SMS Remote Control for Android (Coming Soon)

Control any Android device remotely by sending SMS commands from authorized numbers. No internet connection required -- just cellular network.

What makes xCtl different:

- Full device control over SMS with 50+ commands including location tracking, camera capture (front/back), audio/video recording, screen locking, factory wipe, file management, clipboard access, app install/uninstall, and shell execution
- Whitelist-based security -- only phone numbers you explicitly authorize can send commands. Every command is logged with sender, timestamp, and result
- Telegram bot integration for auto-uploading captured photos, audio, and video recordings directly to your Telegram chat
- Device Admin API support enabling privileged operations like lock, wipe, and password management
- On-device command tester for debugging before relying on SMS
- Built with Flutter for a smooth, modern UI with real-time status monitoring, quick action favorites, and searchable whitelist management

Use cases: lost device recovery, parental monitoring, device management, remote assistance.

Built natively for Android with Flutter + Kotlin. Package: dev.xprime.xctl
